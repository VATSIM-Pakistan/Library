# Data Usage and Handling Policy

## Overview

This policy outlines how VATSIM Pakistan (vACC) collects, uses, stores, and protects personal data of its users. It ensures compliance with applicable data protection standards, including principles aligned with the EU General Data Protection Regulation (GDPR).

This policy applies only to operations within the VATSIM network and does not apply to real-world aviation.

---

## Purpose

The purpose of this policy is to:

- Ensure data protection and privacy compliance  
- Maintain transparency in data usage  
- Protect users and staff  
- Enable smooth operation of VATSIM Pakistan services  

---

## Data Collection

### How We Collect Data

Data is collected through:

- VATSIM API integrations  
- Website services (e.g. feedback, roster, training systems)  
- Discord and support platforms  

By using VATSIM Pakistan services, users consent to the collection and processing of their data.

---

### Types of Data Collected

We may collect:

- Full name and email (from VATSIM account)  
- VATSIM CID and ratings  
- Region, division, subdivision  
- Discord user ID and profile  
- Training and support records  
- Feedback submissions  
- Any additional voluntarily provided data  

---

## Purpose of Data Usage

Data is collected to:

- Manage users and authentication  
- Provide training and support services  
- Maintain ATC rosters  
- Enable access to services  
- Facilitate operational and administrative tasks  
- Verify eligibility for fast-track ratings  

---

## Data Sharing

### Public Data

The following may be publicly visible:

- Name and VATSIM CID  
- Rating and training status  
- Solo validations  
- ATC session bookings  
- Live connection status  

---

### Private Data

Personal data is shared only with:

- VATSIM staff  
- VATWA (regional authority) staff  

No data is shared with unauthorized third parties.

---

## Special Data Handling

### Fast-Track Verification

To verify real-world pilots/controllers:

- Licenses may be submitted  
- Access is restricted to authorized staff only  
- All documents are deleted after verification  

Users are responsible for any incorrect submissions.

---

## Data Usage for Enforcement

Collected data may be used to:

- Investigate misconduct  
- Enforce VATSIM Code of Conduct  
- Apply disciplinary actions where necessary  

---

## Data Storage & Security

- Data is stored in secure systems  
- Access is restricted based on role  
- Access is monitored and audited  
- Encryption and secure protocols are used where applicable  

---

## Data Retention

- Data is retained only as long as necessary  
- Users may request updates or corrections  
- Data may be removed upon valid request  

---

## User Rights

Users have the right to:

- Access their data  
- Request corrections  
- Request deletion where applicable  
- Object to data processing  

Requests should be made through official VATSIM channels.

---

## Policy Updates

- This policy is reviewed periodically  
- Updates are announced via official channels (Discord, forums, website)  
- The latest version will always be available on the website  

---

## Contact

For questions regarding this policy, contact VATSIM Pakistan staff through official communication channels.
